# 期末專題展示
## 專題名稱：🌦️ NutriWeather

## 📌 專題簡介
**NutriWeather** 是一個結合「天氣資訊」與「營養／飲食建議」的智慧型網頁應用。  
系統會根據即時天氣狀況，提供使用者更符合當下環境的飲食參考，協助使用者做出更健康、實用的飲食選擇。

## 🛠️ 使用技術
- Frontend：HTML / CSS / JavaScript
- API：OpenWeather API / Spoonacular API / Calorie Ninjas API / Gemini API / My Memory API
- Deployment：Vercel
- 資料儲存：Google sheet / Apps Script

---

## ✨ 主要功能
- 根據天氣狀況與個人資訊生成三餐食譜
- 飲水與飲食狀況記錄
- 營養價值查詢
- AI健康顧問
